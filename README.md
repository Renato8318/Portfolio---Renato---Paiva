# Portfolio - Renato Paiva

Repositório do meu portfolio pessoal desenvolvido com **React.js**. O projeto apresenta minha trajetória como Desenvolvedor Front-end, minhas competências em automação com Python e análise de dados, além de detalhar os principais projetos que desenvolvi.

## 🚀 Tecnologias e Ferramentas

### Front-end
- **React.js** (Hooks, Router)
- **JavaScript (ES6+)**
- **CSS3** (Variáveis, Flexbox, Grid, Animações)
- **AOS** (Animate On Scroll)
- **React Icons**

### Automação & Dados
- **Python**
- **Web Scraping** (Selenium, BeautifulSoup)
- **Data Analysis** (Pandas)

### Outros
- **Git & GitHub**
- **PostgreSQL / SQL**
- **Salesforce** (Monitoramento de KPIs)

## 📂 Projetos em Destaque

*   **Veritime**: Sistema de acompanhamento de ponto eletrônico em tempo real, focado em jornada de trabalho e controle de pausas.
*   **SessãoPlay**: Aplicação de streaming robusta integrada com a TMDB API para exibição dinâmica de metadados de filmes e séries.
*   **Amicão**: Plataforma social dedicada à adoção de animais, com foco em performance e design centrado no usuário.

## 🛠️ Funcionalidades do Portfolio

- **Modo Escuro/Claro**: Alternância dinâmica de tema com persistência no `localStorage`.
- **HUD Overlay**: Efeito tecnológico nos cards de projetos.
- **Responsividade**: Interface adaptável para dispositivos móveis e desktop.
- **Animações Fluidas**: Uso de AOS e efeitos de brilho (glow) que seguem o cursor do mouse.
